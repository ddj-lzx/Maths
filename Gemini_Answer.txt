Part I: The Integral Summation & The Rational Collapse

1. Identification of f(x):
   Note that A_n can be viewed as the real part of a complex integral:
       A_n = Re[ ∫₀^∞ t^{2n} * e^{-(1-i)t} dt ]
   Using the Gamma function property Γ(z) = ∫₀^∞ t^{z-1} e^{-t} dt:
       A_n = Re[ (2n)! / (1-i)^{2n+1} ]

   Substituting A_n into the series f(x):
       f(x) = Re[ ∑_{n=0}^∞ ((-1)^n * x^{2n}) / (1-i)^{2n+1} ]
       f(x) = Re[ (1/(1-i)) * ∑_{n=0}^∞ (-x^2 / (1-i)^2)^n ]

   The geometric series sum is 1 / (1 - r), where r = -x^2 / (-2i) = -ix^2 / 2:
       f(x) = Re[ (1+i)/2 * (1 / (1 + ix^2/2)) ]
       f(x) = Re[ (1+i) / (2 + ix^2) ]

   By multiplying the conjugate (2 - ix^2):
       f(x) = Re[ ((2+x^2) + i(2-x^2)) / (4 + x^4) ]
       f(x) = (x^2 + 2) / (x^4 + 4)

2. Proof of K = π/2:
   Using partial fraction decomposition:
       (x^2 + 2) / (x^4 + 4) = (1/2) * [ 1/((x-1)^2+1) + 1/((x+1)^2+1) ]
   
   Integrating from 0 to ∞:
       K = (1/2) * [ arctan(x-1) + arctan(x+1) ]₀^∞
       K = (1/2) * [ (π/2 - (-π/4)) + (π/2 - π/4) ]
       K = (1/2) * [ 3π/4 + π/4 ] = π/2.  (Q.E.D.)
