Solution: The Veiled Oasis

Part I – The Mirage of Integrals

1. 计算 A_n 的显式

考虑积分:
I_n = ∫_0^∞ t^{2n} e^{-t/2} cos( (√3/2) t ) dt
    = Re ∫_0^∞ t^{2n} e^{-(1/2 - i√3/2)t} dt

令 α = 1/2 - i√3/2 = e^{-iπ/3}, 则 Reα = 1/2 > 0, 由 Gamma 函数:
∫_0^∞ t^{2n} e^{-α t} dt = (2n)! / α^{2n+1}

由于 α = e^{-iπ/3}, 故 1/α^{2n+1} = e^{i(2n+1)π/3}. 因此
I_n = Re[ (2n)! e^{i(2n+1)π/3} ] = (2n)! cos((2n+1)π/3).

由定义 A_n = 4 I_n, 即得
A_n = 4(2n)! cos((2n+1)π/3)

2. 求 f(x) 的闭形式

方法一 (复几何级数法 – 拉玛努金式洞察):
将 A_n 代入级数:
f(x) = ∑_{n=0}^∞ (-1)^n [4(2n)! cos((2n+1)π/3)]/(2n)! x^{2n}
     = 4 ∑_{n=0}^∞ (-1)^n cos((2n+1)π/3) x^{2n}.

利用 cosθ = Re(e^{iθ}), 交换求和与取实部:
f(x) = 4 Re[ e^{iπ/3} ∑_{n=0}^∞ ( -e^{i2π/3} x^2 )^n ]
     = 4 Re[ e^{iπ/3} / (1 + e^{i2π/3} x^2) ]

计算实部:
e^{iπ/3} / (1 + e^{i2π/3} x^2)
= (1/2 + i√3/2) / (1 + (-1/2 + i√3/2) x^2)
= (1 + i√3) / (2 - x^2 + i√3 x^2) · (2 - x^2 - i√3 x^2)/(2 - x^2 - i√3 x^2)
= ( (1 + i√3)(2 - x^2 - i√3 x^2) ) / ( (2 - x^2)^2 + 3x^4 )

分母: (2 - x^2)^2 + 3x^4 = 4 - 4x^2 + x^4 + 3x^4 = 4 - 4x^2 + 4x^4 = 4(1 - x^2 + x^4)
分子实部: Re[(1 + i√3)(2 - x^2 - i√3 x^2)] = 1·(2 - x^2) + √3·(√3 x^2) = 2 - x^2 + 3x^2 = 2 + 2x^2 = 2(1+x^2)

因此
f(x) = 4 · [ 2(1+x^2) ] / [ 4(1 - x^2 + x^4) ] = 2(1+x^2) / (1 - x^2 + x^4)

方法二 (生成函数法 – 从递推到代数方程):
记 a_n = 4(-1)^n cos((2n+1)π/3), 则 f(x) = ∑_{n=0}^∞ a_n x^{2n}.
由三角恒等式
cos((2n+3)π/3) + cos((2n-1)π/3) = 2 cos(2π/3) cos((2n+1)π/3) = - cos((2n+1)π/3),
乘以 (-1)^n 并注意到 (-1)^{n+1} = -(-1)^n, (-1)^{n-1} = -(-1)^n, 可得系数递推关系:
a_{n+1} + a_{n-1} = a_n   (n ≥ 1).
初始值: a_0 = 4 cos(π/3) = 2, a_1 = -4 cos π = 4.
设 F(t) = ∑_{n=0}^∞ a_n t^n, 利用递推导出方程:
F(t) - a_0 - a_1 t = t(F(t)-a_0) - t^2 F(t),
整理得 (1 - t + t^2) F(t) = a_0 + (a_1 - a_0)t = 2 + 2t.
于是 F(t) = 2(1+t)/(1-t+t^2). 代回 t = x^2 即得
f(x) = F(x^2) = 2(1+x^2)/(1 - x^2 + x^4).
此结果与方法一完全一致.

3. 计算 K = ∫_0^∞ f(x) dx

首先将 f(x) 部分分式分解:
2(1+x^2)/(1 - x^2 + x^4) = 1/(x^2 + √3 x + 1) + 1/(x^2 - √3 x + 1).

方法一 (初等积分):
∫_0^∞ dx/(x^2 + √3 x + 1)
= ∫_0^∞ dx/[(x+√3/2)^2 + 1/4]
= [2 arctan(2x+√3)]_0^∞
= 2(π/2 - arctan√3) = 2(π/2 - π/3) = π/3.

∫_0^∞ dx/(x^2 - √3 x + 1)
= ∫_0^∞ dx/[(x-√3/2)^2 + 1/4]
= [2 arctan(2x-√3)]_0^∞
= 2(π/2 - arctan(-√3)) = 2(π/2 + π/3) = 5π/3.

因此
K = π/3 + 5π/3 = 2π

方法二 (复变围道积分):
因为 f(x) 是偶函数, K = 1/2 ∫_{-∞}^∞ f(x) dx.
考虑 g(z) = 2(1+z^2)/(1 - z^2 + z^4), 在上半平面内有两个一级极点:
z₁ = e^{iπ/6}, z₂ = e^{i5π/6}.
计算留数:
Res(g, z₁) = 2(1+z₁²)/(4z₁³ - 2z₁) = 2(1+e^{iπ/3})/(4e^{iπ/2} - 2e^{iπ/6}) = 2(1+1/2+ i√3/2)/(4i - √3 - i) = (3+ i√3)/(-√3+3i) = -i,
Res(g, z₂) = 2(1+z₂²)/(4z₂³ - 2z₂) = 2(1+e^{i5π/3})/(4e^{i5π/2} - 2e^{i5π/6}) = 2(1+1/2- i√3/2)/(4i + √3 - i) = (3- i√3)/(√3+3i) = -i.
由留数定理,
∫_{-∞}^∞ f(x) dx = 2πi × (-i - i) = 2πi × (-2i) = 4π,
故 K = 1/2 × 4π = 2π.

Part II – The Quintic Spring

由 Part I 知 K = 2π, 则
2π/K = 2π/(2π) = 1
原方程化为
y^5 + 5y^3 + 5y = 1

代数代换 (拉玛努金式洞察):
设 y = z - 1/z, 利用恒等式
y^5 + 5y^3 + 5y = z^5 - z^{-5}.
(验证: 展开 (z-1/z)^5 并加上 5(z-1/z)^3+5(z-1/z) 后中间项全部抵消, 仅剩 z^5 - z^{-5}.)
于是方程变为
z^5 - 1/z^5 = 1
令 u = z^5, 则 u - 1/u = 1, 即 u^2 - u - 1 = 0, 解得
u = (1 ± √5)/2
取正根 u = (1+√5)/2 (以保证 z 为实数), 则
z = ( (1+√5)/2 )^{1/5},  1/z = ( (√5-1)/2 )^{1/5}.
因此唯一实数解为
y = ( (1+√5)/2 )^{1/5} - ( (√5-1)/2 )^{1/5}
